========================================

== CARA MENJALANKAN PROJECT "phpcrud" ==
========================================



1. Install XAMPP terlebih dahulu. Silahkan download di :

-> https://www.apachefriends.org/download.html



2. Aktifkan XAMPP. Caranya :

-> Klik start > xampp > xampp control panel

-> Start Apache service
-> Start MySQL



3. Letakkan folder "phpcrud" ini ke :

-> C:/xampp/htdocs/(letakkan disini)



4. Import database, tabel, beserta kontennya terlebih dahulu. Caranya :

-> Buka browser dan ketik di url : localhost/phpmyadmin/

-> Klik tab Import pada phpmyadmin
-> Upload file SQL yang ada di folder phpcrud/asset/sql/incgrup.sql

-> Klik Go



5. Kitik pada url :
-> localhost/phpcrud
